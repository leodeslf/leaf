## Terser

```bash
# Install
npm i -g terser
```

```bash
# Use
terser [options] <file>
```

Options:

- `-c`, `--compress`
- `-m`, `--mangle reserved=[reserved-words]`
- `-o`, `--output <file>`
- `--comments [all|false]`
- `--keep-classnames`
- `--keep-fmanes` (keep function names)

More at [CLI Usage](https://terser.org/docs/cli-usage).
