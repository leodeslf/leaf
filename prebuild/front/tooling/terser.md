## Terser

```bash
# Install
npm i -g terser
```

```bash
# Use
terser [option] <file>
```

Options:

- `--comments <all|false>`
- `-c`, `--compress`
- `--keep-classnames`
- `--keep-fnames` (keep function names)
- `-m`, `--mangle reserved=[reserved-words]`
- `-o`, `--output <file>`

----

Reference: [CLI Usage (terser.org)](https://terser.org/docs/cli-usage).
