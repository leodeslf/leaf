# Jekyll CLI

```bash
# Install
gem install jekyll bundler
```

```bash
# Use
jekyll new <project-name>
```

```bash
# To fix 3.0.0+ Ruby
bundle add webrick
```

```bash
# Bundle
bundle exec jekyll build
bundle exec jekyll serve
# `bundle exec` is optional after the first use.
```

More at [Command Line Usage](https://jekyllrb.com/docs/usage/).
