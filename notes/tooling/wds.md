# Web Dev Server

```bash
# Install
npm i -D @web/dev-server
```

```bash
# Use
web-dev-server [option]
```

```bash
# Shorthand
wds [option]
```

Options:

- `-c`, `--config` (for a **web-dev-server.config** file)
- `-h`, `--hostname`
- `--node-resolve`
- `-o`, `--open`
- `-p`, `--port <number>`
- `-r`, `--root-dir <dir>`
- `-w`, `--watch`

More at [CLI flags](https://modern-web.dev/docs/dev-server/cli-and-configuration/#cli-flags).
