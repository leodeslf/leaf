# Create React App

```bash
npx create-react-app <project-name>
```

```bash
npm start
npm run build
```

```json
// package.json
{
  "scripts": {
    "start": "set PORT=<number> && react-scripts start",
    "build": "react-scripts build"
  }
}
```

More at [Quick Start](https://create-react-app.dev/docs/getting-started#quick-start).
