# Vue

```bash
npm i -g @vue/cli
vue create <project-name>
```

```bash
npm run serve
npm run build
```

---

```json
// package.json
{
  "scripts": {
    "serve": "vue-cli-service serve --port <number> --open",
    "build": "vue-cli-service build"
  }
}
```
