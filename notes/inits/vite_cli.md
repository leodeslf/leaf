# Vite CLI

```bash
# Initialize
npm init vite@latest <project-name> --template <option>
```

Template options:

- `vanilla`
- `vue`
- `react`
- `preact`
- `lit-element`
- `svelte`

Use `<option>-ts` for TypeScript.

```bash
# Use
vite --port <number> --open
vite build
vite preview --port <number>
```

```bash
# Scripts
npm run dev
npm run build
npm run preview
```

More at [Command Line Interface](https://vitejs.dev/guide/#command-line-interface).
