# 11ty

```bash
# Install
npm i -g @11ty/eleventy
```

```bash
# Use
eleventy [option]
```

Options:

- `--input [dir]`
- `--output <dir>`
- `--quiet <boolean>`
- `--watch`
- `--serve`
- `--port <number>`
- `--formats [format-name]`
- `--config <config-file>`

```bash
# Use with npx
npx @11ty/eleventy [option]
```

More at [COMMAND LINE USAGE](https://www.11ty.dev/docs/usage/).
