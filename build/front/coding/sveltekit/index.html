SvelteKit

# Initialize
npm init svelte@next <project-name>
# Use
svelte-kit dev --port <number> --open
svelte-kit build
svelte-kit preview --port <number> --open
# Use with npm scripts
npm run dev
npm run build
npm run preview

Adding SASS/SCSS

npm i -D svelte-preprocess node-sass
// svelte.config.js
import preprocess from 'svelte-preprocess';

const config = {
  preprocess: preprocess({
    scss: {}
  })
};

export default config;

More at:

  • scss, sass (github.com).
  • Svelte Preprocess (github.com).

Adding a Vercel adapter

npm i -D @sveltejs/adapter-vercel
// svelte.config.js
// import adapter from '@sveltejs/adapter-auto'; // By default.
import vercel from '@sveltejs/adapter-vercel';

const config = {
  kit: {
    adapter: vercel()
  }
};

export default config;

More at:

  • adapter (kit.svelte.dev).
  • Adapters (kit.svelte.dev).
  • adapter-vercel (github.com).

Reference: Getting started (kit.svelte.dev).
