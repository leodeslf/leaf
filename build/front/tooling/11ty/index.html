11ty

# Install
npm i -g @11ty/eleventy
# Use
eleventy [option]
# Use with npx
npx @11ty/eleventy [option]

Options:

  • --config <config-file> (.eleventy.js by default)
  • --formats [format-name]
  • --input [dir]
  • --output <dir>
  • --port <number>
  • --quiet <boolean>
  • --serve
  • --watch

Reference: COMMAND LINE USAGE (11ty.dev).
