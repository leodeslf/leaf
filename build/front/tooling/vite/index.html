Vite

# Initialize
npm init vite@latest <project-name> --template <option>

Template options:

  • vanilla
  • vue
  • react
  • preact
  • lit-element
  • svelte

Use <option>-ts for TypeScript.

# Use
vite --port <number> --open
vite build
vite preview --port <number>
# Use with npm scripts
npm run dev
npm run build
npm run preview

Reference: Command Line Interface (vitejs.dev).
