Web Dev Server

# Install
npm i -D @web/dev-server
# Use
web-dev-server [option]
# Shorthand
wds [option]

Options:

  • -c, --config <config-file> (web-dev-server.config by default)
  • -h, --hostname
  • --node-resolve
  • -o, --open
  • -p, --port <number>
  • -r, --root-dir <dir>
  • -w, --watch

Reference: CLI flags (modern-web.dev).
