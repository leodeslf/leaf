Terser CLI

npm i -g terser
terser [options] <file>

Options:

  • -c, --compress
  • -m, --mangle reserved=[<reserved-words>]
  • -o, --output <file>
  • --comments [all | false]
  • --keep-classnames
  • --keep-fmanes (keep function names)
