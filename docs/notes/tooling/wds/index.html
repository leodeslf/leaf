Web Dev Server

# Install
npm i -D @web/dev-server
# Use
web-dev-server [option]
# Shorthand
wds [option]

Options:

  • -c, --config (for a web-dev-server.config file)
  • -h, --hostname
  • --node-resolve
  • -o, --open
  • -p, --port <number>
  • -r, --root-dir <dir>
  • -w, --watch
