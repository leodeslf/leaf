Web Dev Server

npm i --save-dev @web/dev-server
npx wds --help
web-dev-server [options]
# or
wds [options]

Options:

  • -c, --config (for a web-dev-server.config file)
  • -r, --root-dir
  • -o, --open
  • --node-resolve
  • -w, --watch
  • -p, --port
  • -h, --hostname
// package.json
{
  "scripts": {
    "start": "wds -p <number> --node-resolve -w -o"
  }
}
