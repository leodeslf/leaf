# SASS

## Installation

```bash
npm i -D sass
```

## Usage

```bash
sass [option] <input-file> <output-file>
sass [option] [<input-file>:<output-file>]
```

Options:

- `-c`, `--color`
- `--no-source-map`
- `-q`, `--quiet`
- `--update` (compiles only modified files)
- `-w`, `--watch`

More at [Dart Sass Command-Line Interface (sass-lang.com)](https://sass-lang.com/documentation/cli/dart-sass).
