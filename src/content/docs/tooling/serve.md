# Serve

## Installation

```bash
npm i -g serve
```

## Usage

```bash
serve <dir> [option]
```

Options:

- `-C`, `--cors` (sets `Access-Control-Allow-Origin: *`)
- `-p <number>` (port)
- `-s`, `--single` (404 redirects to `index.html`)
- `-u` `--no-compression`

More at [Usage (github.com)](https://github.com/vercel/serve#usage).
