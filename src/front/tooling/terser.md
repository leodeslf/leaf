# Terser

```bash
# Install
npm i -g terser
```

```bash
# Use
terser [options] <file>
```

Options:

- `--comments <all|false>`
- `-c`, `--compress`
- `--keep-classnames`
- `--keep-fnames` (keep function names)
- `-m`, `--mangle reserved=[reserved-words]`
- `-o`, `--output <file>`

More at [CLI Usage](https://terser.org/docs/cli-usage).
