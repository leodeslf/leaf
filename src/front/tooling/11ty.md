# 11ty

```bash
# Install
npm i -g @11ty/eleventy
```

```bash
# Use
eleventy [option]
```

```bash
# Use with npx
npx @11ty/eleventy [option]
```

Options:

- `--config <config-file>` (.eleventy.js by default)
- `--formats [format-name]`
- `--input [dir]`
- `--output <dir>`
- `--port <number>`
- `--quiet <boolean>`
- `--serve`
- `--watch`

----

Reference: [COMMAND LINE USAGE (11ty.dev)](https://www.11ty.dev/docs/usage/).
