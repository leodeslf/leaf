# Vue

```shell
# Initialize (new)
npm init vue@next
```

```bash
# Install (old)
npm i -g @vue/cli
```

```bash
# Use (old)
vue create <project-name>
vue-cli-service serve --port <number> --open
vue-cli-service build
```

```bash
# Scripts
npm run serve
npm run build
```

More at [CLI Service](https://cli.vuejs.org/guide/cli-service.html#using-the-binary).
